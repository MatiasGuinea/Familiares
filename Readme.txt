Proyecto Familiares:
-------------------

1) Posee una aplicacion que gestiona 3 entidades (Tablas) independientes (No estan relacioandas.  
	- Familiares
	- Amigos
	- Mascotas

2) La aplicacion tiene 4 opciones de menu, donde en 3 de ellas se pueden visualizar datos de cada una de estas tablas y agregar registros a traves de un formulario, y la 4ta opcion esta asociada a la busqueda de registros por nombre en la tabla de mascotas.

3) El motor de BD es SQLite3.
